# Menu-Driven Calculator (C++)

A simple console-based calculator that performs Addition, Subtraction,
Multiplication, and Division through a repeating menu.

## Files
- `calculator.cpp` — the full source code

## How to Compile

Using g++ (Linux/macOS/Windows with MinGW):
```
g++ -o calculator calculator.cpp
```

## How to Run

Linux/macOS:
```
./calculator
```

Windows:
```
calculator.exe
```
